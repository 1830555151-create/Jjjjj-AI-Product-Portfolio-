## 🚀 Project 1: Architectural Spatial Control via AIGC
**Objective:** Standardize AI visual outputs for structural comparative experiments.

* **The Challenge:** When generating architectural models, AI often struggles with scale and proportion.
* **The Workflow:** I conducted a comparative experiment between a semi-circular arch and a pointed arch. Initially, the AI generated images of mismatched proportions.
* **Prompt Intervention:** By enforcing strict pixel-level constraints and explicit spatial positioning commands, I successfully forced the model to generate both arch types at the exact same size, allowing for a rigorous academic comparison.
